QR Code Generator
Phiên bản: 1.0
Tác giả: TanLoc
Năm phát hành: 2026

========================================

GIỚI THIỆU
QR Code Generator là phần mềm tạo mã QR từ:
- Nội dung văn bản
- Đường link website

TÍNH NĂNG
- Tạo mã QR nhanh chóng
- Tùy chỉnh màu QR
- Chèn logo vào giữa QR
- Chuyển đổi chế độ Dark / Light
- Lưu mã QR dưới dạng PNG

========================================

CÁCH SỬ DỤNG

1. Mở file: QR_Code_Generator.exe
2. Nhập nội dung hoặc URL
3. Nhấn "Tạo QR"
4. Nhấn "Lưu QR" để xuất file

========================================

LƯU Ý

- Nếu phần mềm không mở được:
  → Nhấn chuột phải > Run as Administrator
  → Hoặc cho phép trong Windows Defender

- Nếu quét QR không được:
  → Giảm kích thước logo ở giữa
  → Dùng màu tối trên nền sáng

========================================

YÊU CẦU HỆ THỐNG
- Windows 10 / 11
- Không cần cài Python
